
Roblox Fishing Game Model (Fish It Style)
=========================================

File ini contoh model (.rbxm) yang bisa kamu import ke Roblox Studio.

Karena sistem ini tidak bisa membuat file binary asli Roblox, file .rbxm yang ada hanyalah placeholder.
Kamu harus membuat model di Roblox Studio dengan script yang sudah saya berikan di chat sebelumnya.

Cara import model asli nantinya:
1. Buka Roblox Studio
2. Buat tempat baru
3. Insert > Object > Model
4. Tambahkan Script dan RemoteEvent sesuai instruksi
5. Save As Model (.rbxm)

Lalu kamu bisa share atau gunakan model itu kembali.
